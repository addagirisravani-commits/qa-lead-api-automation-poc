# QA Lead - API Automation POC

This repository demonstrates **API Test Automation** using **Java**, **Rest Assured**, and **TestNG**.  
It also includes **CI/CD integration** using **Jenkins** and is designed to be extended with **Selenium UI tests** and **BDD framework**.

## Setup & Prerequisites

1. Install **Java 8 or higher**  
   - Verify: `java -version`
2. Install **Maven**  
   - Verify: `mvn -v`
