# QA Lead - Automation POC

This repo contains API Test Automation using Java + Rest Assured + TestNG.

Objective of this repo:
- To demonstrate end-to-end automation capability
- To show CI/CD integration using Jenkins
- To highlight QA lead skills in modern automation approach

## Tech Stack

| Area       | Used |
|------------|------|
| Language   | Java 8+ |
| Build Tool | Maven |
| API Test   | Rest Assured |
| Test Runner| TestNG |
| CI/CD      | Jenkins Pipeline |

## How to run tests

```bash
mvn clean test
